---
name: jira-ticket
description: Retrieves the title and description of a JIRA ticket. Use when a user asks about a "jira ticket" or "JIRA ticket".
---

Use this skill to retrieve the details of a JIRA ticket.

Fetch the details of the ticket from jira using the `jira` CLI, e.g. `jira issue view abc-123 --plain` (where `abc-123` is the ticket number), then get the output
